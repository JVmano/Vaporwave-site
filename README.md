# Vaporwave-site
Trabalho oferecido pelo Professor Jorge do curso técnico de Informática no Colégio Estadual do Paraná.

# O que é?
O site se consiste em uma landing page de uma equipe de e-sports fictícia (em partes) com um design focado essencialmente em Vaporwave, um estilo retrô que ganhou muita força nos últimos anos, devido à sua grande versatilidade e utilização por mídias gerais.

Idealizado e desenvolvido por João Vitor de Freitas Melo
